# 🏢 오늘 회사에서 살아남기 (Survive the Office)

> 직장인의 하루 업무를 HP(체력 게이지)로 기록하는 유쾌한 생존 대시보드

🔗 **[라이브 데모 보기](https://survive-the-office.vercel.app)**

---

## 🎮 서비스 소개

**"오늘 살아서 퇴근할 수 있을까?"**

회의, 긴급 요청, 메일 폭탄… 매일 에너지를 갉아먹는 업무들.  
반면 커피 한 잔, 팀 칭찬, 맛있는 점심이 하루를 버티게 만들기도 하죠.

*오늘 회사에서 살아남기*는 이런 업무 이벤트를 **HP(체력 게이지)** 로 기록하고,  
오늘의 컨디션을 **날씨**로 시각화해주는 직장인 생존 대시보드입니다.

퇴근 시 **생존 등급(S~F)** 을 확인하고 팀과 공유하세요.

---

## 🎯 해결하는 문제

> "직장인의 번아웃은 갑자기 오지 않는다. 매일 조금씩 쌓인다."

기존 업무 도구는 **할 일 관리**에 집중하지만, **감정·에너지 소모**는 기록되지 않습니다.  
오늘 회사에서 살아남기는 하루의 **심리적 에너지 흐름**을 가시화해 자기 인식을 돕고,  
팀원과 가볍게 공유함으로써 **심리적 안전감**을 높이는 것을 목표로 합니다.

**타겟 사용자**: 번아웃을 경험했거나 예방하고 싶은 20~40대 직장인  
**차별점**: 무드 트래커와 달리 업무 이벤트 기반의 HP 시스템 + 팀 공유 기능

## ✨ 핵심 기능

| 기능 | 설명 |
|---|---|
| ☀️ **업무 날씨 대시보드** | HP 잔량에 따라 맑음 → 폭풍으로 실시간 변화 |
| 💊 **이벤트 기록** | 버튼 클릭 한 번으로 HP 증감 + 툴팁 애니메이션 |
| 🏆 **퇴근 결과 판정** | 하루 종료 시 S~F 생존 등급 + 등급별 애니메이션 |
| 👥 **팀 공유 보드** | 팀원들의 오늘 날씨/HP 상태 한눈에 확인 + 응원 기능 |
| 📅 **히스토리 캘린더** | 날짜별 생존 기록 조회 및 월별 트렌드 분석 |
| ➕ **커스텀 이벤트** | 나만의 이벤트 직접 추가 가능 |

## 🎮 HP 이벤트 예시

**➕ 회복 이벤트**
- ☕ 커피 한 잔 `+8` / 🍱 점심 맛있었음 `+12` / 👏 칭찬 받음 `+15`
- 🎉 프로젝트 성공 `+20` / 🍩 간식 등장 `+10`

**➖ 데미지 이벤트**
- 📧 메일 폭탄 `-10` / 🔥 긴급 요청 `-15` / 🌀 야근 확정 `-20`
- 😤 불필요한 회의 `-15` / 🤯 멀티태스킹 `-10`

## 🌤️ 날씨 & HP 매핑

| HP | 날씨 | 상태 메시지 |
|---|---|---|
| 81~100 | ☀️ 맑음 | "오늘 무적 모드! 뭐든 가능해요" |
| 61~80 | ⛅ 구름 조금 | "순항 중. 이 기세 유지!" |
| 41~60 | ☁️ 흐림 | "버티는 중. 커피 한 잔 어때요?" |
| 21~40 | 🌧️ 비 | "우산 챙기세요. 오늘 좀 빡세네요" |
| 1~20 | ⛈️ 폭풍 | "전멸 직전! 간식이 필요한 순간" |
| 0 | 💀 전멸 | "오늘은 여기까지… 수고했어요" |

> **시작 HP는 80** — 출근과 준비 과정의 기본 피로도를 반영합니다.

## 🤖 AI-Native 개발 방식

이 프로젝트는 **Claude(Anthropic)** 를 활용한 AI-Native 개발 방식으로 제작되었습니다.

| 단계 | AI 활용 내용 |
|---|---|
| 설계 | PRD·CLAUDE.md·디자인 토큰 구조 설계 |
| 구현 | Claude Code로 전체 컴포넌트 구현 및 scaffolding |
| 디버깅 | 타입 오류·CSS 변수 불일치·레이아웃 문제 해결 |
| 검수 | validation_plan 체크리스트 항목별 코드 검수 |
| 개선 | 사용성 테스트 결과 기반 반복 개선 |

AI가 일관된 코드를 생성할 수 있도록 [`CLAUDE.md`](./CLAUDE.md)에 도메인 개념, 인터랙션 명세, 디자인 토큰을 명시적으로 정의했습니다.
